# Smoke Test Checklist

- [ ] Build succeeded.
- [ ] Package exists.
- [ ] `DwmLutGUI.exe` starts as Administrator.
- [ ] `dwm_lut.dll` is beside the EXE.
- [ ] Apply succeeds.
- [ ] `%SystemRoot%\Temp\dwm_lut.log` exists.
- [ ] Log contains selected DWM profile.
- [ ] Log contains staged monitor manifest.
- [ ] Disable succeeds.
