# Notice

This repository is a maintained fork of the original DWM LUT project.

Original project:

https://github.com/lauralex/dwm_lut

Forward-port base used for this work:

https://github.com/zkippp/dwm_lut_fixed

Original purpose:

Apply 3D LUTs to the Windows desktop through DWM for system-wide color correction and calibration.

This fork focuses on continued compatibility for modern Windows 11 builds, including 25H2, 26H1, 26H2 Experimental, and Canary/Future Platforms where possible.
