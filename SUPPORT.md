# Support

Before opening an issue, collect:

- Windows version and exact build number
- Windows channel: Stable, Release Preview, Beta, Dev, Canary, or Future Platforms
- GPU model
- GPU driver version
- Monitor model
- HDR enabled: yes/no
- Multi-monitor setup: yes/no
- LUT format and size
- `%SystemRoot%\Temp\dwm_lut.log`

Unsupported reports:

- "It does not work" without a build number
- Calibration accuracy claims without measurement data
- Reports from modified binaries without source diff
- Reports without logs for injection failures
